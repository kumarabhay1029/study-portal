📋 Submission Guidelines

📄 Format Requirements

  • All assignments must be submitted in PDF format
  • Include proper cover page with student details
  • Use standard fonts (Arial 12pt or Times New Roman 12pt)
  • Maintain proper page margins (1 inch on all sides)

💻 Programming Assignments

  • Submit source code with proper comments
  • Include screenshots of program execution
  • Provide detailed explanation of algorithm used
  • Test with multiple input scenarios

⏰ Submission Process

  • Submit assignments through the portal before deadline
  • Late submissions may result in grade penalties
  • Keep backup copies of all submitted work
  • Verify submission confirmation email
